# K61

Мои авторские колонки. 

Закрытый Ящик 9 литров. Peerless P830883, Vifa DX25BG60-04, если это может о чем-то сказать.

# Кабинеты

Изготовлены из 12мм МДФ. Распорки есть. Синтепон внутри есть, где-то по 100 грамм в каждом корпусе. Разъем GX20 4pin.

![1.JPG](/photos/1.JPG)
![2.JPG](/photos/2.JPG)

# Усилитель

TPA3250 2 штуки, ADAU1701 1 штука. БП на 24 и 5 вольт здесь же. Вход - TRS стерео. Выход - GX20 4pin 2 штуки. Разъем питания - как в компьютерных БП. Все собрано на модулях с алиэкспресса, по заветам колхозного DIY.

Громкость во время измерений была максимальная (line out у focusrite 2i2 2 gen), по ощущениям ДБ 85, прибора нет. Все измерения лежат в `./measures`.
